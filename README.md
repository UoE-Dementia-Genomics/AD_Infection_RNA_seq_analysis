# ADInfection_transcriptomics_analysis
## AD Infection – RNA-seq and Integrative Analysis Pipeline

This repository contains an R pipeline to analyse bulk RNA-seq data and integrate it with co-expression networks, pathway activity, enrichment analyses and single-cell reference datasets in the context of Alzheimer’s disease (AD) and respiratory infection.

## Dependencies

All scripts are written in R and rely heavily on Bioconductor.
